# Fami Tech Application

## Prerequisites

- Node.js ([v16.16.0](https://nodejs.org/ja/blog/release/v16.16.0) or higher)

## Getting started

### 1. Clone this repository

```bash
$ git clone <REPO_URL>
```

### 2. Install dependencies

```bash
$ cd client

$ npm install

$ cd ..
$  npm install
```

### 3. Run application locally

```bash
$ npm run app
```
